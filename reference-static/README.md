# nexs.tudio Website

Static launch site for nexs.tudio.

## Files

- `index.html` - the website page structure
- `styles.css` - responsive visual styling and motion
- `script.js` - mobile nav, section tracking, reveal animation, and email form behavior
- `google-flow-prompts.md` - video prompts for Google Flow
- `assets/` - current brand images copied from Downloads

## Video Placement

After generating Google Flow videos, add them here:

- `assets/video/hero-flow.mp4`
- `assets/video/product-morph.mp4`

The site already references those paths. Until the videos exist, the brand poster images are shown.
